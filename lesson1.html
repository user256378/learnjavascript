Головна / Вступ

Початок

Javascript (скорочено JS) - основна мова програмування Всесвітньої павутини, яка дозволяє визначити поведінку веб-сторінок. В JS можна взаємодіяти з користувачами, реагувати на події, отримувати і використовувати дані з Інтернету, виводити графіку і т.д.

Нагадаємо!

HTML використовується для визначення контенту сторінок та їх структури. CSS-кольори, шрифти, рамки і визначають стиль оформлення сторінок.

JS - родич HTML i CSS і вміє визначити поведінку веб-сторінок

Програміст включає код JS прямо в сторінку, а потім завантажує її в браузер. Браузер сам виконує усе необхідне для виконання коду.

Код JS складається з команд. Кожна команда описує частину операції що буде виконуватися, а весь набір команд визначає поведінку сторінки.

Змінні і значення

Змінні призначені для зберігання значень.

Розглянемо приклади:

var winners = 2;// команда оголошує змінну winners і присвоює їй числове значення 2var name = "Alex";// зміннй надається послідовність символів (рядок)var log = false;// змінні що приймають значення true i false називають логічними (булевими)

Крім чисел, рядків і логічних значень змінні можуть зберігати і інші дані. Незалежно від виду даних усі змінні створюються по одним правилам:

var winners = 2;

Запам'ятайте!

  • Оголошення змінної завжди починається з ключового слова var
  • Після цього пишемо ім'я змінної (як правильно давати ім'я змінній?)
  • При бажанні вказується значення змінної після знака =
  • Команда присвоєння завжди закінчується знаком ;

Можна створити змінну без початкового значення: 

var winners;

Отже, у змінної є ім'я і є значення.

Як правильно давати ім'я змінній?

  • Ім'я змінної має починатися з літери, підкреслення або знака доллара.
  • Потім можуть йти літери, підкреслення або знаки доллара - в будь-якій кількості.
  • Не можуть використовуватися вбудовані ключові слова.
  • У JS регістр символів враховується.
  • Ім'я змінної не має містити пробіл.

Синтаксис

  • Кожна команда закінчується символом ;
  • Однорядковий коментар починається з двох похилих рисок (//)
  • Зайві пропуски дозволяються (майже всюди)
  • Рядки беруться в подвійні лапки (або одинарні)
