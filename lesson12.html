Дії з атрибутами

Зробимо текст: "Red Alert" червоним тривожним кольором.

Визначимо для класу "redtext" правило CSS, що надає абзацу червоний колір тексту.

Додамо код, який бере елемент абзацу "greenplanet" і додає до нього клас redtext.

Об’єкти element містять метод setAttribute, який може викликатися для завдання атрибутів елементів HTML. Метод використовується для додавання чи зміни вже існуючого атрибута.

planet.setAttribute("class", "redtext");

Метод отримує два аргументи:

  • ім’я атрибута
  • значення атрибута

Значення атрибута можна читати методом getAttribute. 

var scoop = document.getElementById("raspberry");var altText = scoop.getAttribute("alt");console.log("I can not see the image in the console,");console.log("but i'm told it looks like: " + altText);

Якщо шуканий атрибут не існує в елементі то ми отримаємо null.

Перевірка: 

var scoop = document.getElementById("raspberry");var altText = scoop.getAttribute("alt");  if (altText == null) {  console.log("No here an alt attribute");} else {  console.log("I can not see the image in the console,");  console.log("but i'm told it looks like: " + altText);}

Що ще можна робити з моделлю DOM?

  • Отримувати елементи (всі елементи класу "onsale", значення введені користувачем в формі).
  • Створювати і додавати елементи в DOM.
  • Вилучати елементи.
  • Перебирати елементи (дочірні даного, сусідні).
