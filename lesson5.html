1 спосіб

if (inStock == true) {  if (onSale == true) {    alert("Buy!");  }}

Випадкове розміщення кораблів

Для того щоб вірно розмістити корабель нам потрібна початкова позиція, від якої будуть відраховуватися три суміжні клітинки. Отже, початкова клітинка має належати діапазону від 0 до 4.

Якщо у нас є початкова позиція randomLoc (від 0 до 4), то:

var location1 = randomLoc;var location2 = location1 + 1;var location3 = location2 + 1;

Math.random() - функція, яка повертає випдкове число діапазону від 0 до 1 (не включаючи 1, наприклад 0,128..).

var randomLoc = Math.random();

Якщо домножимо випадкове число на 5 то отримаємо число з [0;5). Залишилось прибрати дробову частину.

var randomLoc = Math.floor(Math.random() * 5);

Math.floor() - функція, яка округлює число, прибираючи дробову частину.

Є ще одна помилка – якщо тричі ввести одне й те саме число, то корабель потоне.
