Що можна робити з рядками

Властивість length повертає кількість символів у рядку (зручно використовувати для перебору символів у рядку). 

var text = "hello";console.log(text.length);

Метод charAt() отримує ціле число від 0 до довжини рядку (мінус 1) і повертає рядок з одним символом, що знаходиться в заданій позиції. Рядок можна рознлядати як масив, елементами якого є окремі символи, а індекси починаються з 0. Якщо переданий індекс більше або дорівнює довжині рядку, повертаєтсья порожній рядок. 

var input = "jenny@wickedlysmart.com";for (var i = 0; i < input.length; i++) {  if (input.charAt(i) == "@") {    console.log("There's an @ sign at index" + i);  }}

Метод indexOf() отримує рядок-аргумент і повертає індекс першого символу першого входження аргумента в своєму рядку. 

var phrase = "The cat in the hat";var index = phrase.indexOf("cat");console.log("There's a cat sitting at index " + index);

Також можна передати другий аргумент - індекс початкової позиції пошуку. 

var phrase = "The cat in the hat";var index = phrase.indexOf("cat", 5);console.log("There's a cat sitting at index " + index);

Якщо рядок знайти не вдалось, метод повертає -1.

Метод lastIndexOf() знаходить останнє входження підрядка в рядку.

Метод substring() отримує два індекса, після чого витягує і повертає рядок між ними (не включаючи 2й індекс). 

var data = "name|phone|address";var val = data.substring(5, 10);console.log("Substring is " + val);

Другий індекс можна не вказувати, в цьому випадку substring дістане рядок, який починається з першого індекса і продовжується до кінця рядку. 

var data = "name|phone|address";var val = data.substring(5);console.log("Substring is " + val);

Метод split() отримує символ-обмежувач і розбиває рядок на частини по позиції обмежувача. 

var data = "name|phone|address";var vals = data.split("|");console.log("Split array is " + vals);

В цьому випадку масив vals не перетворюється в рядок перед виведенням в консоль, тому в console.log передається два аргумента, розділених комою.

Метод trim() вилучає пропуски з обоx кінців рядку. 

var text = "hello";var newtext = text.trim();console.log("New text is " + newtext);
