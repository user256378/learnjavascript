Вирази та обчислення

Розглянемо вираз у наступній команді:

Чому буде дорівнювати змінна total, якщо price = 10, discount = 20?

var total = price - (price * discount / 100);

Існують і рядкові вирази:

"Dear" + "Reader" + ","// конкатенація

або

"super" + "cali" + youKnowTheRest

Результатом обчислення також може бути істиною чи хибою (true або false). Такі вирази називають булевими (або логічними).

age < 14
cost >= 3.99
animal == "bear"// істина, якщо назва тварини bear

Результатом обчислення виразу буде число, рядок або логічний вираз.

Отже:

= - оператор присвоювання

== - оператор перевіряє чи рівні два значення

!= - оператор перевіряє що два значення НЕ рівні

Цикл while

В програмах операції доводиться повторно виконувати. Для цього у JS використовуються конструкції while, for, for in, for each.

Розглянемо приклад:

while (scoops > 0) {  document.write("Another scoop!");  scoops = scoops - 1;}

while - ключове слово
scoops > 0 - умова
В дужках { } - програмний блок

Прийняття рішень в JS (умовний оператор if)

Розглянемо приклад:

if (scoops < 3) {  alert("Ice cream is running low!");}

if - ключове слово
scoops < 3 - умова
В дужках { } - програмний блок команди if

Функція alert отримує рядок і виводить його у спливаючому вікні браузера.

Команда if дозволяє послідовно виконувати декілька перевірок:

if (scoops >= 5) {  alert("Eat faster, the ice cream is going to melt!");} else if (scoops == 3) {  alert("Ice cream is running low!");} else if (scoops == 2) {  alert("Going twice!");} else if (scoops == 1) {  alert("Going once!");} else if (scoops == 0) {  alert("Gone!");} else {  alert("Still lots of ice cream left, come get it");}

Функція prompt отримує рядок, виводить його у спливаючому вікні браузера та чекає, поки користувач введе текст, а потім повертає введене значення або null, якщо введення скасоване (Cancel/ESC). Результат роботи функції prompt є типом string, тобто рядком. 

var age = prompt("How old are you?");alert(age);

Якщо потрібен результат типу number, тобто число, необхідно використати +prompt замість prompt: 

var age = +prompt("How old are you?");alert(age);
